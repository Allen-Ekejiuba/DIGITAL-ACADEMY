PROJECT OVERVIEW
----------------
This is a fully responsive, multi-page educational website for the
Digital Skills Training Academy (DSTA), built with HTML5, CSS3,
and vanilla JavaScript.

FOLDER STRUCTURE
----------------
academy-website/
│
├── index.html          → Home Page
├── about.html          → About Page
├── courses.html        → Courses Page
├── contact.html        → Contact Page
│
├── css/
│   └── style.css       → Main stylesheet (all styles, responsive)
│
├── images/
│   ├── banner.jpg          → Hero banner image
│   ├── about-hero.jpg      → About page hero image
│   ├── team.jpg            → Team/instructors image
│   ├── webdev.jpg          → Web Development course image

│   ├── data.jpg            → Data Science course image
│   ├── cybersecurity.jpg   → Cybersecurity course image

PAGES & FEATURES
----------------
index.html (Home)
  - Sticky navigation header with logo
  - Hero section with statistics
  - Features strip
  - Featured Courses section (3 course cards)
  - Why Choose Us (4 benefit cards)
  - External Resources section (3 external links)
  - CTA banner
  - Footer with links, social icons

about.html (About)
  - Page hero with breadcrumb
  - About intro with image
  - Mission & Vision cards
  - Expert Instructors team grid
  - Core Values section
  - External partner links

courses.html (Courses)
  - Filter bar (visual)
  - 6 full course cards with:
    images, descriptions, tags, meta, pricing, enrol links
  - CTA banner

contact.html (Contact)
  - Contact info panel (address, phone, email, hours)
  - Social media links (6 platforms)
  - Contact form with validation
  - Map placeholder
  - FAQ section

HTML FEATURES USED
------------------
✅ Semantic tags: <header>, <nav>, <main>, <section>, <article>, <footer>
✅ Navigation between all 4 pages
✅ External links (freeCodeCamp, Coursera, MDN, W3Schools, NBTE, Google for Startups)
✅ 10+ images with descriptive alt attributes
✅ Various image sizes used throughout
✅ Breadcrumb navigation on inner pages
✅ Accessible form with labels and required fields

CSS FEATURES USED
-----------------
✅ CSS Custom Properties (variables)
✅ Flexbox layouts
✅ CSS Grid layouts
✅ Custom color palette
✅ Background images with overlays
✅ Borders and rounded corners
✅ Box shadows
✅ Hover transitions and animations
✅ Responsive: mobile, tablet, desktop
✅ Mobile hamburger menu (via JavaScript)

RESPONSIVE BREAKPOINTS
----------------------
Desktop:  1200px max-width container
Tablet:   ≤ 960px — 2-column grids
Mobile:   ≤ 640px — single column, hamburger nav
Small:    ≤ 480px — further adjustments

HOW TO VIEW
-----------
1. Open index.html in any modern web browser
2. Navigate between pages using the top navigation
3. All links and images use relative paths — no server required
